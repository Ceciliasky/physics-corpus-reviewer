## 第十六章 电压 电阻

我国西部地区的水能和煤炭资源丰富，可以用来大量发电。为了把这些电能输送到数千千米以外电力紧缺的东部地区，同时减少损耗，人们采用了一种叫作特高压输电的方式。2018年，我国建成了从新疆昌吉至安徽古泉的目前世界上电压等级最高、输送容量最大、输送距离最远、技术最先进的特高压直流输电工程。

通过这一章的学习，我们将了解电压是什么，它有什么作用。

### 第1节 电压

问题

通过上一章的学习我们知道，要使用电器能够工作，电路中必须有电源。电源是如何使电流持续流过用电器的呢？

让我们类比水流的情形来认识这个问题。

电压

在图16.1-1所示的装置中，容器A中的水位高于容器B。打开阀门，就会形成由A经过水管流向B的水流。要想让水管中的水连续不断地流动，就必须使容器A中的水位一直高于容器B，即保持A、B之间的水压。我们可以用抽水机不断地把低水位容器中的水抽到高水位容器中去，来保持水压。抽水机是提供水压的装置。

图16.1-2 电源给用电器提供电压

与此类似，要使电荷在电路中定向移动，电路的两端就必须要有电压（voltage）。电源的作用就是给用电器两端提供电压（图16.1-2），从而能形成持续的电流。

方法点拨

水压使水定向运动，从而形成水流。电流和水流都可以看作某种物质的“流动”，根据这一相似性我们可以推测：电压使电荷定向移动，从而形成电流。这里利用了类比的方法。

不同的电源可提供不同的电压，一节干电池提供的电压远低于家庭电路的电压。通常用字母  $U$  表示电压，它的单位是伏特（volt），简称伏，符号是  $\mathrm{V}$  。

当电压较高时，常用千伏（ $\mathrm{kV}$ ）作单位；当电压较低时，常用毫伏（ $\mathrm{mV}$ ）作单位。它们的换算关系是

$$
1 \mathrm {k V} = 1 0 0 0 \mathrm {V} = 1 0 ^ {3} \mathrm {V}
$$

$$
1 \mathrm {m V} = 0. 0 0 1 \mathrm {V} = 1 0 ^ {- 3} \mathrm {V}

在上面的实验中，电池提供的电压是相同的。当电路中接入不同的导体时，电流表的示数通常不同。为什么会有这种差别呢？

原来，导体虽然容易导电，但是对电流也有一定的阻碍作用。在物理学中，用电阻（resistance）来表示导体对电流阻碍作用的大小。导体的电阻越大，表示导体对电流的阻碍作用越大。导体的电阻通常用字母  $R$  表示，单位是欧姆（ohm），简称欧，符号是  $\Omega$  。比较大的单位有千欧（ $\mathrm{k}\Omega$ ）、兆欧（ $\mathrm{M}\Omega$ ）。它们之间的换算关系是

$$
1 \mathrm {k} \Omega = 1 0 0 0 \Omega = 1 0 ^ {3} \Omega
$$

$$
1 \mathrm {M} \Omega = 1 0 0 0 0 0 0 \Omega = 1 0 ^ {6} \Omega
$$

图16.3-3 常用的电阻器

实验室小灯泡中灯丝的电阻通常为几欧姆到十几欧姆；日常用的电炉丝的电阻为几十欧姆；长  $1\mathrm{m}$  、横截面积  $1\mathrm{mm}^2$  的家用铜导线，电阻为百分之几欧姆，通常可以略去不计。

我们经常要用到具有一定电阻的元件——电阻器（图16.3-3），它也叫作定值电阻，在电路图中用符号—□—表示。

2. 超导现象

各种金属导体中，银的导电性能是最好的，但还是有电阻存在。20世纪初，荷兰物理学家昂内斯发现，当温度降低到某个临界值时一些金属的电阻会突然消失，这种现象叫作超导现象。零电阻和完全抗磁性是超导体的基本特性。

变阻器的应用

滑动变阻器一般只在实验室中应用。有些家用电器中，用于音量调节的器件也是一种变阻器，通常称为电位器。图16.4-5是一种电位器的结构图。它通过机械式旋钮调节电阻的大小。电位器也可以用在其他电器上，例如可调亮度的电灯，可调音量的音箱、调音台，等等。

除机械式电位器外，数字电位器的使用越来越广泛。数字电位器是一种用数字信号控制电阻的器件（集成电路）。与机械式电位器相比，数字电位器具有可用程序控制改变电阻、耐震动、噪声小、寿命长、抗环境污染等优点，因而已在自动检测与控制、智能仪器仪表、消费类电子产品等许多重要领域得到应用（图16.4-6）。
